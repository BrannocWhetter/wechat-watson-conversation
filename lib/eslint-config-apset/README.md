# eslint-config-apset
Node Style Guide for PET.

## Install

```bash
npm i eslint @apset/eslint-config-apset
```

OR

```bash
yarn add eslint @apset/eslint-config-apset
```

## Usage
* Create `.eslintc` file
```json
{
  "extends": "eslint-config-apset"
}
```
